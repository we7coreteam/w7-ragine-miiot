Rangine Framework
一款基于Swoole开源高性能异步&协程开发框架。基于Swoole扩展，常驻内存，不依赖传统的php-fpm。

文档系统

基于软擎开发的开源版文档管理系统，是微擎多年开发经验的技术沉淀产品。支持多个编辑器，多用户协同操作，管理员审核发布等功能。让您的工作更高效，更智慧。

积分商城

基于软擎开发的开源版积分商城系统，是微擎为拓展业务需求所成立的自建商城新项目，可用于兑换商品或服务，支持入驻，积分+钱等系统化的积分运营，有助于拉近您与客户的关系，提高付费转化比例。

协程/异步

原生的异步和协程支持，支持HTTP、TCP/UDP、Webso- cket等协议，可以满足高并发场景要求。

组件化

符合Laravel框架和PSR的规范，实现了路由、日志、缓存常用组件，可以让你简单、优雅、快速的实现功能。

学习成本低

引入Laravel框架的DB门面及Eloquent模型，拥有超低的学习成本。

安装环境的准备及注意事项

不要忽略任何一处微小的细节

环境要求

PHP > 7.0.0、PHP Swoole 扩展 > 4.2.3、PHP Mbstring 扩展、PHP Pdo MySql

安装

通过 Composer 来管理依赖，请确保你的机器上安装了 Composer

git clone https://github.com/we7coreteam/w7-rangine-empty ./rangine-test && cd rangine-test && sudo composer install
